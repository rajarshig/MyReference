
## Basics

- Start
```
screen -S [screen_view_name]
```
- Entered into screen view. now run command
eg. python script 

- Detach
```
screen -d res_gid
```
- Re attach
```
screen -d [name]
```
- List all running Screen
```
screen -ls
```

- Ctrl-a followed by d: detach a screen session without stopping it
- ctrl-a followed by k: kill session
